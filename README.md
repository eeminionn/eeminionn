# Emilio Abarca

Interaction Design student at **Universidad del Desarrollo (UDD)** building across software and the physical world: robots, connected devices, fabricated interfaces and audiovisual experiments.

These repositories contain firmware, workshop material, fabrication logs, tests and design documentation. Implementation details stay close to the artifacts: pinouts near firmware, machine settings near fabricated parts and controls near interactive code.

## Community

- Founder of **EIRI UDD**, an interdisciplinary robotics and innovation team.
- Mentor at **Fundacion Mustakis**, supporting robotics and STEAM learning experiences.
- Contributor to workshop guides, technical documentation and student-facing prototypes at UDD.
